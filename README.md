# ruoyi-maximize
为Ruoyi等项目制作的全屏组件，用于解决浏览器 API 无法直接全屏内部页面的问题。实现覆盖菜单栏并扩展页面内容至整个视口，提供沉浸式浏览体验。

# 文件结构
fullscreen
├─ fullscreen.vue         # 封装好的全屏切换组件（内置放大/缩小图标）
├─ FullscreenToggle.vue   # 可传入自定义图标的全屏切换组件
├─ index.vue              # 全屏组件调用示例
├─ magnify.png            # 放大图标（进入全屏）
└─ shrink.png             # 缩小图标（退出全屏）
